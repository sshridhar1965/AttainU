1.	git pull  	-  Used to download latest version of any repository from Github to Local
2.	git push  	-  To upload any repository from Local to remote.
3.	git merge 	-  to merge any 2 branches into one
4.	git log   	-  to view a history of all the commits
5.	git status	-  to get status about staged or unstaged
6.	git commit	-  to save the changes in local repository to Remote
7.	git init	-  to declare any local repository to Git , first time
8.	git clone	-  to make a copy of remote repository to local
9.	git branch 	-  to make a branch from any commit on the main/master
10.	git checkout	-  to move to head of any other brach from main/master.
